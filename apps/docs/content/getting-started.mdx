# Getting Started

This library was built to ease the development of websites for AOM.

<Alert color="warn">This project is still under devlopment and is subject to change.</Alert>

## Installation

<CodeBlock value="npm i @aomdev/ui" fileType="Terminal" fileName="Terminal">
```bash
npm i @aomdev/ui

````
</CodeBlock>

You will also need to install [Tailwind](https://tailwindcss.com/docs/installation) for component styles.
